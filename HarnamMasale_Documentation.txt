## Introduction
The Harnam Masale website is a complete e-commerce solution for a premium Indian spice brand. The website features responsive design, user authentication, shopping cart functionality, product browsing, and order management. The site uses Firebase for backend services including authentication, database, and real-time updates.

## Project Overview
- **Project Name:** Harnam Masale
- **Type:** E-commerce Website
- **Target Audience:** Indian spice consumers
- **Year Developed:** 2025
- **License:** See LICENSE file

## Technology Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Firebase (Authentication, Realtime Database)
- **Hosting:** Firebase Hosting
- **External Libraries:**
  - Font Awesome (v6.4.0) for icons
  - Google Fonts (Poppins)

## Key Features
1. **Responsive Design:** Fully responsive website that works on all devices
2. **Theme Support:** Dark and light theme options
3. **User Authentication:** Login, signup, and profile management
4. **Shopping Cart:** Add/remove products, adjust quantities
5. **Product Catalog:** Browsable product listings with filtering
6. **Checkout Process:** Multi-step checkout with address input
7. **Order Management:** View order history and status
8. **Contact Form:** Customer inquiry submission
9. **Dynamic Content:** Firebase-powered product data
10. **Animations:** Smooth transitions and interactive elements

## User Interface
The website features a modern, clean interface with:
- Responsive navigation menu with mobile sidebar
- Consistent header and footer across pages
- Card-based product displays
- Form validations for user inputs
- Loading animations and transitions
- Theme toggle for user preference

## Authentication System
The authentication system is implemented using Firebase Authentication with custom UI:
- Email/password authentication
- Guest user support with temporary IDs
- User profile management
- Password reset functionality
- Session management
- Secure routes that require authentication

## E-commerce Functionality
The e-commerce system includes:
- Product browsing with filters
- Shopping cart management
- Order processing
- Payment method selection (COD implemented, card/UPI coming soon)
- Shipping address collection
- Order confirmation
- Order history tracking

## Firebase Integration
Firebase provides backend services:
- **Authentication:** User login and registration
- **Realtime Database:** Product data, user profiles, orders
- **Cloud Storage:** Product images and assets
- **Hosting:** Website deployment

## Responsive Design
The website is designed with a mobile-first approach:
- Fluid layouts that adapt to screen size
- Optimized images for different devices
- Touch-friendly interface elements
- Mobile sidebar navigation
- Responsive typography

## Theme Support
The website supports two themes:
- **Dark Theme:** Default theme with dark backgrounds and light text
- **Light Theme:** Optional theme with light backgrounds and dark text
- Theme preference is saved in local storage
